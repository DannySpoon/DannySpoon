Danny Spoons



This website was created basically as a means of practise for writing HTML and CSS. I wanted to make a simple website that featured the basic elements that most websites incorporate. The code itself is an unstructured mess so I am surprised everything is working so far, hopefully I will continue to build websites and move onto cleaner and more advanced code.

The website has a fixed navigation bar and logo which links to three other pages and includes some minor hover transformations. The content on the body of the pages is all pretty straightforward – either text or images. There is also a footer with a gradient colour property, I am undecided as to whether it looks crap or not.

I have also used this as a way to archive some pictures I have taken while traveling. I also wish to write some general musings in the articles section. The problem with writing HTML and not using a service something like WordPress or Squarespace is that I can’t find an efficient way to add new content to the website in a simple and automated way.
